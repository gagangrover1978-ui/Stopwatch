# Responsive Stopwatch

A clean, responsive stopwatch web application built with vanilla HTML, CSS, and JavaScript. 

## Features
* **Responsive Design:** Uses CSS Grid and Flexbox to ensure the layout looks great on all screen sizes.
* **Accurate Logic:** Built using `setInterval` with formatting for hours, minutes and seconds.
* **Pro Formatting:** Uses `padStart` to maintain the `00:00:00` digital clock format.
